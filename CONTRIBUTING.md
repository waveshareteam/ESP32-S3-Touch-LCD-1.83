# Contributing

Thank you for helping improve this repository.

1. Open an issue or discussion for behavior changes that affect examples, board support, or CI.
2. Keep examples under `examples/esp-idf/` or `examples/arduino/`.
3. Prefer managed ESP-IDF components for reusable board, display, touch, sensor, audio, and video support.
4. Keep bundled Arduino libraries under `examples/arduino/libraries` when sketches depend on the bundled version.
5. Run the relevant ESP-IDF or Arduino build before opening a pull request.

Use repo-relative paths in reports and validation notes.
