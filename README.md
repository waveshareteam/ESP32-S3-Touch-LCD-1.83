## Overview

This repository provides first-party ESP-IDF and Arduino examples, factory
firmware, schematics, video assets, and maintainer documentation for the
Waveshare ESP32-S3-Touch-LCD-1.83 development board.

The board combines an ESP32-S3 with a 1.83-inch IPS LCD, capacitive touch,
power management, motion sensing, and a real-time clock in a compact form
factor.

## Hardware Overview

| Feature | Device / interface |
| --- | --- |
| MCU | ESP32-S3 |
| Display | 1.83-inch 240 x 284 IPS LCD using ST7789 |
| Touch | CST816T capacitive touch controller |
| Power management | AXP2101 |
| Motion sensor | QMI8658 six-axis IMU |
| RTC | PCF85063 |
| Board support | ESP-IDF component `waveshare/esp32_s3_touch_lcd_1_83` |
| Hardware files | [Schematic](schematic/) and [video assets](videos/) |

## Firmware and Recovery

The [Firmware](Firmware/) directory contains checked-in factory firmware for
flashing and recovery. These files are recovery artifacts, not source projects
and are not built by the example workflow.

Source-built ESP-IDF and Arduino firmware is generated by the [Build Examples
workflow](https://github.com/waveshareteam/ESP32-S3-Touch-LCD-1.83/actions/workflows/examples.yml)
when the corresponding CI packaging path is available. Downloaded workflow
artifacts are not committed to the repository.

## Build and CI

Build an ESP-IDF example for the ESP32-S3 target:

    idf.py -C examples/esp-idf/02_lvgl_demo_v9 set-target esp32s3 build

Compile an Arduino example with the bundled libraries:

    arduino-cli compile --fqbn esp32:esp32:esp32s3 \
      --libraries examples/arduino/libraries \
      examples/arduino/01_HelloWorld

The current CI matrix uses ESP-IDF `v5.5.4` and `v6.0.2`, and
Arduino-ESP32 core `3.3.10` with FQBN `esp32:esp32:esp32s3`. CI discovers
first-party examples under `examples/esp-idf/` and `examples/arduino/` while
excluding bundled library examples. See [CI documentation](docs/ci.md) for
the workflow matrix and dispatch options.
